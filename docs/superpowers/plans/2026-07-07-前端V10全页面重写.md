# 前端 V10 全页面重写 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 按 V10 设计模板(`~/文档/V10 Hikami-Go 全页面重设计.html`)全页面重写前端 4 个 view(首页/主播/回顾/设置),替换 Element Plus 为 V10 自定义组件库,以 OpenAPI 契约(`docs/api/openapi.yaml` + `web/src/api/generated.ts`)为唯一类型源。

**Architecture:** 四阶段递进:**Phase 0 后端补字段**(Session/Task 加 `channel_name`、listSessions 支持 `channel_id`/`source`/`search` 过滤)→ **Phase 1 设计系统基座**(CSS 变量 + V10 自定义组件库 `HButton/HCard/HPill/HDrawer/HDialog/HInput/HSelect/HSwitch/HToggle/HCheckbox/HCollapse/HTable/HTabs`)→ **Phase 2-5 逐页重写**(4 view,每页独立分支、可独立 merge、可在任意页暂停)→ **Phase 6 清理**(移除 Element Plus 依赖)。**逐页迁移**:每页一个分支、PR 独立审核、独立回滚。

**Tech Stack:** Vue 3 (Composition API + `<script setup>`) + Pinia + vue-router 4 + Vite + TypeScript + axios + marked + dompurify + mitt + qrcode。**移除** element-plus、@element-plus/icons-vue。

## 全局约束(每个任务都遵守)

1. **类型源唯一**:`web/src/api/generated.ts`(openapi-typescript 生成)。所有新组件的 props/emit 类型、新 API wrapper 的入参出参类型,从 `generated.ts` 的 `paths` / `components['schemas']` 派生,**不再**手写 `web/src/api/types.ts`(Phase 2 起逐步迁移现有类型,Phase 6 删除 `types.ts`)。
2. **保留分层**:`api/`(契约层,无 UI 副作用)、`stores/`(实体缓存)、`composables/`(跨域复用 hook)、`features/`(业务组件)、`components/`(共享展示)、`views/`(路由壳)。**业务逻辑下沉 features,view 退化为壳**(详见 `docs/FRONTEND_ARCHITECTURE.md` §1)。
3. **保留 5 个 store + 刷新协调器**:`useAppRefreshCoordinator` 仍是 WS + 降级轮询唯一 owner。`sessionActions.ts` 状态机不变(41 单测保护)。
4. **V10 设计 token 锁定**:CSS 变量定义见 Phase 1 Task 1.2 `design-tokens.css`,**不得在组件内硬编码颜色/圆角/阴影**,只能引用 `var(--xxx)`。例外:scrollbar/skip-link 等极少纯展示元素。
5. **不动后端 API 路径/方法/payload**:除 Phase 0 显式补字段外,后端接口契约由 OpenAPI 锁定,**前端不得私改 wrapper 的 URL/method/字段名**。
6. **专家模式开关保留**:`useExpertMode` 控制高级配置/运行中任务等区块显隐。
7. **旧路径重定向保留**:router/index.ts 的 9 条 `redirect` 不变(`/live`→`/`、`/sessions/:sid`→`/recaps?sid=` 等)。
8. **每页可独立 merge**:页面间通过 store + router 解耦,任一页迁移完成后即可合并,不阻塞其他页(旧 EP view 与新 V10 view 共存期内,router 按需切换)。

```vue
<!-- web/src/components/ui/HButton.vue -->
<script setup lang="ts">
defineProps<{
  variant?: 'primary' | 'secondary' | 'ghost' | 'danger'
  size?: 'sm' | 'xs' | 'md'  // 默认 md(无 size 类,原 .btn 基类尺寸)
  disabled?: boolean
  loading?: boolean
  type?: 'button' | 'submit'
}>()
const emit = defineEmits<{ click: [event: MouseEvent] }>()
function onClick(e: MouseEvent) {
  // loading/disabled 时阻止冒泡(避免提交表单等副作用)
}
</script>
<template>
  <button
    :type="type ?? 'button'"
    class="btn"
    :class="[variant ? `btn-${variant}` : 'btn-primary', size ? `btn-${size}` : '']"
    :disabled="disabled || loading"
    @click="emit('click', $event)"
  >
    <span v-if="loading" class="btn-spinner" aria-hidden="true" />
    <slot />
  </button>
</template>
```

```ts
// web/src/features/home/composables/useElapsedDuration.ts
import { computed, onUnmounted, ref } from 'vue'

export function useElapsedDuration(getStartedAt: () => string, tickMs = 1000) {
  const now = ref(Date.now())
  const timer = setInterval(() => { now.value = Date.now() }, tickMs)
  onUnmounted(() => clearInterval(timer))

  const text = computed(() => {
    const start = new Date(getStartedAt() || '').getTime()
    if (!start || Number.isNaN(start)) return '-'
    const sec = Math.max(0, Math.floor((now.value - start) / 1000))
    const h = String(Math.floor(sec / 3600)).padStart(2, '0')
    const m = String(Math.floor((sec % 3600) / 60)).padStart(2, '0')
    return `${h}:${m}`
  })

  return { text }
}
```
